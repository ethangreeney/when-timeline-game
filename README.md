# WHEN?

WHEN? is a daily timeline game about surprising historical collisions. Players place eight mystery events between two revealed anchors, with two lives and a new shared puzzle each calendar day.

The game includes deterministic daily puzzles, local streak and progress saving, spoiler-free sharing, sound and reduced-motion support, and unlimited themed practice runs.

Run `npm install` and `npm run dev` for local development. Run `npm test` to lint and build the deployable site.
